# Survey instruments

Questionnaires and field instruments used in AVE-RD. Spanish-first, with
English translations where they exist. These are intended for replication —
researchers in other countries can copy or adapt them under the project's
use license.

## Expected files

| File | Wave | Respondent | Language |
|---|---|---|---|
| `baseline_student_es.pdf` | Baseline (2015) | Student | ES |
| `baseline_student_en.pdf` | Baseline (2015) | Student | EN |
| `baseline_parent_es.pdf` | Baseline (2015) | Parent | ES |
| `baseline_teacher_es.pdf` | Baseline (2015) | Teacher | ES |
| `baseline_principal_es.pdf` | Baseline (2015) | Principal | ES |
| `endline_student_es.pdf` | Endline (2016) | Student | ES |
| `endline_parent_es.pdf` | Endline (2016) | Parent | ES |
| `followup_student_es.pdf` | Follow-up wave | Student | ES |

Add codebooks alongside the questionnaires if they exist (`*_codebook.pdf`
or `*_codebook.xlsx`).
